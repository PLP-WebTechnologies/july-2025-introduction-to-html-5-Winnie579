Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content on the web and is the foundation of all websites.

Common HTML Tags

HTML tags are used to define and structure content on a webpage. Each tag has a specific purpose, whether it's for headings, paragraphs, images, or links. Understanding these basic tags is essential as they form the backbone of any HTML document.

  • <h1> to <h6> - Headings: Define titles or subtitles of content.
  • <p> - Paragraph: Used for blocks of text or descriptions.
  • <a> - Hyperlink: Creates clickable links to other pages or sections.
  • <img> - Image: Embeds images into a webpage using a source (src) URL.
  • <ul> - Unordered List, <ol> - Ordered List, <li> - List Item: Used to create lists of items.
